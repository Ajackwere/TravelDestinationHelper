## A Website To Help Pick the Best Travel Destinations (Hidden Gems) in Kenya.
## By Austine Jack Were.

## Description
An interactive web app to review tourist destinations and also help users make a choice of places to visit.

Regions you can visit
Nairobi
Kisumu
Mombasa
Nakuru

As a user, you will be able to:
view the ratings and comments made by others concerning tourist destinations (the hidden gems) in Kenya.
Add your comments to a destination (both positive and negative comments allowed). 
## Project Setup
1. Clone this repository
2. Make a copy in your local environment.
3. Open the server using the command <json-server --watch db.json>
4. Open index.html in your browser to view the website.
